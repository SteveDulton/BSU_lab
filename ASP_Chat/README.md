Вариант с публикацией:
	1) Открыть: Панель управления->Программы->Прогаммы и компоненты
	2) Открыть: Включение или отключение компонентов Windows 
	3) В появившемся окне найти пункт "Службы IIS" выбрать его 
	
	4) В меню Пуск открыть "Диспетчер служб IIS"
	5) В списке слева нажать ПКМ по узл "сайты" и "обавить веб-сайт..."
	6) Задать произвольное "Имя сайта", в поле "Физический путь" указать C:\inetpub\wwwroot\[Имя_сайта]
	7) Задать Порт и запомнить его!

	8) Открыть брандмауэр Windows->Дополнительные параметры
	9) В списке слева выбрать пункт "Правила для входящих подключений" и кликнуть ПКМ "Создать правило"
	10) Протокол TCP; выбрать порт, который Вы запомнили!; Разрешить подключение; Далее,далее...; (Имя и описание по вкусу)

	11) Содержимое архива Local_Chat.rar перенести в C:\inetpub\wwwroot\[Имя_сайта]
	12) готово (перейти по сайту с того же пк можно через localhost:[указанный порт]
	    или с другого устройства указав [IP_где развёрнуто_приложение]:[указанный порт])
